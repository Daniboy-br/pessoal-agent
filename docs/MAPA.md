# MAPA — docs/

Documentação simples do app Nimbo.

## Pastas

- `setup/` — configuração técnica e Supabase.
- `features/` — mapa funcional das capacidades criadas no produto.

## Regra

Toda feature, módulo, rota, serviço ou tabela relevante criada no Nimbo deve ganhar um material simples em `features/` explicando:

1. o que é;
2. para que serve;
3. como funciona;
4. arquivos/tabelas envolvidos;
5. o que já está pronto;
6. o que falta;
7. como verificar.
