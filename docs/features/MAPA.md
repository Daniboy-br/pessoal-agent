# MAPA — docs/features/

Mapa funcional das capacidades do Nimbo.

Cada arquivo explica uma funcionalidade criada no app, de forma simples e rastreável.

## Features documentadas

- `agent-turns.md` — tabela de turnos do agente e rastreabilidade de runtime.
- `persistencia-onboarding.md` — salvamento do pré-questionário, perfil, Nimbo e agentes no Supabase.
- `agent-turn-service.md` — serviço que orquestra um turno de conversa do guia central.
- `prompt-builder.md` — montagem do prompt por blocos auditáveis.
- `runtime-architecture-v1.md` — separação do runtime em `SessionRouter`, `MemoryService`, `ObservabilityService`, `PromptBuilder v1` e `AgentTurnService` orquestrador.
